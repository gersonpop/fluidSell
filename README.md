# sell-ia monorepo

Estructura del repositorio:

- `web/`: Aplicacion Next.js (frontend + API interna).
- `docs/`: Documentacion funcional y tecnica.
- `mobile/`: App movil.
- `n8n/`: Flujos y automatizaciones de n8n.

## Desarrollo web

```bash
cd web
npm install
npm run dev
```

## Deploy en Vercel

Configurar el proyecto para usar `web/` como **Root Directory**.
