# Scrum Operating Model

## Objetivo

Usar Scrum para desarrollo incremental y trazable.

## Estados

- Backlog
- Ready
- InProgress
- InReview
- Done
- Blocked

## Definition of Ready

- Objetivo claro
- Modulo responsable
- Criterios de aceptacion
- Dependencias conocidas
- Riesgos identificados
- Prueba esperada

## Definition of Done

- Entregable implementado
- Pruebas ejecutadas
- Errores esperados manejados
- Documentacion actualizada
- Sin secretos expuestos
- Trazabilidad en `.scrum/scrum-db.json`
