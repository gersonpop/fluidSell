## ✍️ Tipografía
- **Titulares (Headings):** *Plus Jakarta Sans* o *Outfit* – estilo geométrico, moderno y legible.
- **Cuerpo y UI:** *Inter* – fuente neutra y altamente legible en tamaños pequeños.

---

## 📏 Directrices de Uso
1. **Espaciado:** Mantener un margen mínimo del 20 % del ancho del icono alrededor del mismo cuando se use como favicon o avatar.
2. **Escala:** Utilizar el icono 64 px para interfaces de escritorio, 32 px para barra de navegación y 16 px para favicons.
3. **Colores:** Aplicar la paleta de colores corporativos en cualquier elemento de UI que acompañe al logotipo; evitar combinaciones de bajo contraste (p.ej., coral sobre fondo coral).
4. **Fondo:** Cuando se use el logotipo principal, emplear fondos oscuros (`#0C0E12`) para resaltar la versión blanca; en fondos claros, usar la versión en negro.
